# Creating & Managing Posts

## Create New Post - Step by Step

**Navigate to**:
1. Go to /user/posts
2. Click "Create New Post" button

**Fill form**:
```
Title: "My First Blog Post"
Slug: (auto-generates: my-first-blog-post)
Excerpt: "A brief summary of the post"
Content: (Markdown content here)
Category: Select from dropdown
Tags: Type and add multiple tags
SEO Title: (for search engines)
SEO Description: (for search preview)
Options: Check "Featured" if important
```

**Save**:
- "Save Draft" → Save unpublished
- "Publish" → Save and make visible

**After creating**: Auto-redirect to post list

---

## Edit Existing Post

**Access**:
1. Go to /user/posts
2. Find post in list
3. Click "Edit" button

**Changes**: Modify any field (title, content, category, status, etc.)

**Save**: Click "Update" to save changes

---

## Publish/Archive Posts

**From User List** (/user/posts):
```
Draft post → Click "Publish" → Becomes visible
Published post → Click "Archive" → Moves to archived
```

**From Admin** (/dashboard/posts):
```
Select multiple draft posts → Click "Publish" button → Bulk publish
Select multiple posts → Click "Archive" → Bulk archive
```

---

## Delete Post

**User Area**:
1. Go to /user/posts
2. Click "Delete" on post row
3. Confirm deletion

**Admin Area**:
1. Go to /dashboard/posts
2. Select post (checkbox)
3. Click "Delete" button
4. Confirm bulk deletion

⚠️ **Warning**: Deletion is permanent, cannot undo

---

## Search & Filter Posts

**User List**:
- Filter by status: All/Draft/Published/Archived
- Search box: By title
- Results update in real-time

**Admin List**:
- Filter tabs: All/Draft/Published/Archived
- Search: By title OR author name
- More granular filtering available

---

## Bulk Actions (Admin Only)

**Select Multiple**:
1. Check box next to each post
2. Or use "Select All" checkbox (if available)

**Actions**:
- "Publish All" → Publish all selected
- "Archive All" → Archive all selected
- "Delete All" → Delete all selected

**Confirmation**: Required before executing
